<h1 align="center">Projeto Baduk Back-end Jr</h1>

## Descrição

<p> REST API desenvolvida como resolução de um desafio backend</p>

## Rotas
 - OpenAPI
    
    `[GET]` /docs

## Como executar

```bash
# development
$ yarn start

# build
$ yarn build

# production mode
$ yarn start

# tests 
$ yarn test
```

## Banco de dados

 ```bash
 # Execute all migrations
 $ yarn typeorm:migration:run

 # Run postgres
 $ docker-compose up -d
 ```
